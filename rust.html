General History

Mozilla developed in 2010 Rust, a modern computer programming language, to build high-programming applications without some issues developers faced using C or C++. It's a system programming language that allows smooth integration of computer hardware and software. Its main purpose in performance, safety and memory management. Those elements make Rust a perfect language to develop advanced applications like gaming engines, operating systems and browsers. For example, Firefox and Dropbox both use Rust.
